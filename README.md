# Projek-Akhir-APD

Toko Apotek Nayla adalah program apotek sederhana yang menggunakan python. Pengguna bisa membuat akun atau login, lalu masuk sebagai user atau apoteker. User dapat melihat dan membeli obat, termasuk menjadi member untuk mendapat diskon, sedangkan apoteker bisa menambah, mengubah, menghapus, serta mencari dan memfilter obat. Semua data disimpan di file data.py, proses pengelolaan obat dilakukan di handler.py, dan main.py mengatur menu, tampilan, serta alur utama program agar semuanya berjalan dengan rapi dan interaktif.
